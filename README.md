# Task Manager Frontend (Lab 4) 

##  Про проєкт
Це клієнтська частина системи управління завданнями.
Розроблена в рамках лабораторної роботи №4 з використанням сучасного стеку технологій.

##  Технології
- **Core:** React 18, TypeScript, Vite
- **Styling:** Tailwind CSS, clsx
- **Routing:** React Router v6
- **HTTP:** Axios (з налаштованими інтерцепторами)
- **Code Quality:** ESLint, Prettier

##  Функціонал
- Адаптивний дизайн (Mobile/Desktop).
- Компонентна архітектура (Button, Card, Input).
- Налаштований роутинг (Home, About, 404).
- Layout-система (Header, Footer).
- Темна тема (підготовка в Tailwind).
